## 🚀 À propos de PyNav

PyNav est une application Android permettant de :
- **Afficher** les instruments NMEA (vitesse, cap, profondeur, vent, moteur, jauges...) sur tablette
- **Contrôler** un pilote automatique pypilot (Raspberry Pi)

Disponible sur **Google Play Store** — iOS App Store bientôt.
