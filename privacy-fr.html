3. DONNÉES COLLECTÉES

3.1 Données que nous NE collectons PAS

L'Application ne collecte ni ne transmet à nos serveurs :
  • ❌ Vos positions GPS ou coordonnées géographiques
  • ❌ Vos trajets, routes ou waypoints de navigation
  • ❌ Vos données NMEA (vitesse, cap, profondeur, vent, etc.)
  • ❌ Vos paramètres de configuration pypilot
  • ❌ Vos préférences d'affichage ou tableaux de bord
  • ❌ L'historique de vos navigations
  • ❌ Les commandes envoyées au pilote automatique
  • ❌ Toute information personnelle identifiable (nom, adresse, téléphone)

3.2 Données techniques collectées

Connexion réseau locale (ne quitte PAS votre bateau) :
  • Adresse IP locale de votre serveur pypilot (ex: 192.168.1.100)
  • Port de connexion TCP/IP
  • Ces données restent strictement sur votre réseau local WiFi/Bluetooth et ne sont jamais transmises à l'extérieur
Identifiant d'achat Google Play :
  • Si vous achetez la version PRO, Google Play génère un jeton unique pour vérifier votre achat
  • Ce jeton est géré par Google, pas par nous
  • Nous n'avons accès qu'à la confirmation que vous avez acheté la version PRO
  • Aucune information de paiement n'est accessible à l'éditeur
Données de crash et d'erreurs (optionnel) :
  • Si vous acceptez, l'Application peut envoyer des rapports d'erreurs anonymes via Firebase Crashlytics
  • Ces rapports contiennent : type d'erreur, modèle d'appareil, version Android, version de l'Application
  • Ces rapports sont totalement anonymes et ne contiennent aucune donnée de navigation
  • Vous pouvez désactiver cette fonctionnalité dans les paramètres de l'Application

3.3 Permissions Android requises

L'Application demande les permissions suivantes : INTERNET (obligatoire) :
  • Nécessaire pour se connecter à votre serveur pypilot via WiFi
  • Aucune connexion Internet externe n'est établie sauf pour la vérification d'achat Google Play
ACCESS_NETWORK_STATE (obligatoire) :
  • Permet de détecter si une connexion réseau est disponible
  • Nécessaire pour afficher un message d'erreur si le WiFi est déconnecté
WAKE_LOCK (optionnel) :
  • Empêche l'écran de se mettre en veille pendant l'utilisation
  • Utile pour garder l'Application visible en navigation
BILLING (optionnel) :
  • Nécessaire uniquement si vous souhaitez acheter la version PRO
  • Géré entièrement par Google Play
ACCESS_FINE_LOCATION (optionnel) :
  • Permet d'afficher votre position sur une carte locale si activée
  • Cette position est lue depuis les données NMEA de votre équipement, pas depuis le GPS du téléphone
  • Aucune position n'est jamais transmise à l'extérieur
